## Description

A webpage for a science museum that contains:

- Events page
- Visit Page
- Get involved page

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:atlehjelmseth/semester-project1-csm.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
